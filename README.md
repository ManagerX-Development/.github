# ManagerX Development

## Was wir machen

ManagerX Development entwickelt moderne Management-Software für Discord Communties.

## Team

| Rolle | Aufgaben |
|-------|----------|
| **Team-Leader** | Projektplanung, Architektur |
| **Backend Developer** | API, Datenbank |
| **Frontend Developer** | UI/UX |
| **Tester/QA** | Qualitätssicherung |

## Workflow

### Branches

- `main` – Produktion
- `dev` – Entwicklung
- `feature/*` – Neue Features
- `bugfix/*` – Bugfixes

### Regeln

- Keine Commits auf `main`
- Code-Review erforderlich
- Tests schreiben
- Features dokumentieren

### Commits

Format: `feat: Beschreibung` oder `fix: Beschreibung`
